# LLM Career Mentor Bot

A personal AI mentor to help you become a top-tier AI Engineer. Upload your resume, get daily goals, personalized resources, project guidance, and interview prepâ€”all powered by local LLMs (Ollama) and Streamlit.

## Features
- Daily goal setting
- Personalized learning and resources
- Project guidance (e.g., build a multimodal agent)
- Interview Q&A
- File analysis and feedback

## Tech Stack
- Python, Streamlit, Ollama

## Getting Started
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start Ollama and ensure a model (e.g., llama3) is available.
3. Run the app:
   ```bash
   streamlit run app.py
   ```
